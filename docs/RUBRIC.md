# Interview Rubric

Each major score must include transcript-backed evidence.

## Categories (1-10)

- Technical Proficiency: 1-10
- Communication Clarity: 1-10
- Ownership/Initiative: 1-10
- Codebase Alignment: 1-10

## Red Flags

Examples:
- Contradictory or unverifiable claims
- Lack of ownership in project outcomes
- Repeated inability to explain technical decisions
- Poor collaboration or accountability indicators

Each red flag should include:
- label
- severity
- evidence citation from transcript

## Overall Match Score (0-100)

- Derived from weighted category scores
- Must align with strengths, growth areas, and red flags
- Evidence from transcript required for each major score
