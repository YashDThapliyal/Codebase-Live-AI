# Frontend Generation Notes

This frontend was generated as a scaffold to accelerate delivery for a backend/AI-focused team.

## Intent

- Provide a clean, minimal, modern baseline
- Avoid over-engineering or deep design work early
- Keep components small and easy to replace

## Team Guidance

- Humans should avoid spending too much time on styling
- Candidate UI should remain simple and non-intimidating
- Admin UI should remain functional and readable
- Use mock-first flows to preserve development speed

## Scope Discipline

Defer complex frontend work unless it is required for the demo or blocks integration.

Prefer:
- clear information hierarchy
- readable tables/cards
- obvious extension points

over:
- advanced state architecture
- animation-heavy polish
- speculative component abstractions
